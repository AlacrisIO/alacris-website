  • Problem

    The biggest obstacle with blockchain is nobody actually “uses” blockchain. Users want apps that make their lives better. They don’t care about the tech behind them. However, valuable apps come from a critical mass of developers building many, many apps. For this to happen, developers need to get past the difficult learning curve for blockchain tech and be confident in writing bug free code.

  • Solution

    The Alacris OS platform allows for sophisticated, end-to-end DApp development, including automatic formal verification of the code for any protocol or platform. Alacris offers easy-to-use primitives as modular, flexible starting blocks that any developer can apply.

Alacrity

A DSL for Trustworthy Distributed Applications Cascading verification through type theory, theorem proving, model checking, strand spaces, and dynamical system simulation The promises our tools provide
  • Execution: Guarantee that on-blockchain behavior respects DSL semantics.
  • Full Abstraction: Guarantee that all DSL semantics cannot be violated by direct blockchain interactions.
Expressiveness through end-point projection: one source, many programs Verification of DSL Programs…
  • Correctness: Guarantee that the state space of the protocol matches the desired functional characteristics, like a balanced ledger.
  • Security: Guarantee that only authenticated participants influence the protocol behavior (i.e., non-interference of adversaries) in the Dolev-Yao model.
  • Efficiency: Guarantee that the state evolves in Pareto-efficient way for all protocol-following participants.
